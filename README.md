# CapCut Clone App

A cross-platform mobile video editing application built with Flutter (Frontend) and FastAPI (Backend).

## Project Structure

This project is set up as a monorepo:

- **`frontend/`**: Flutter application for iOS and Android.
- **`backend/`**: FastAPI backend server built with Python, providing user authentication, cloud storage, project sync, and AI video processing services.

- **Frontend Setup**: See [frontend/README.md](frontend/README.md)
